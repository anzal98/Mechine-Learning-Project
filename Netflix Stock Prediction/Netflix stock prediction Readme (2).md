
# Netflix Stock Prediction


This project uses machine learning techniques to predict the opening value of Netflix stock for the next day. The prediction model is trained on historical data from 2015 to 2019, including high, low, and closing values of the stock.

### Getting Started
To get started with this project, you will need to have Python installed on your computer. You will also need to install the necessary libraries and dependencies, such as scikit-learn and pandas.

### Data
The data used in this project consists of historical stock prices for Netflix from 2015 to 2019. This data includes high, low, and closing values for each day.

### Model
The prediction model is trained using machine learning technique 'lienear Regression'. Various techniques are explored and evaluated to determine the most effective approach for predicting Netflix stock prices.
